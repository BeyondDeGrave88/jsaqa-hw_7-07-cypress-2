# Cypress & Jenkins

Проект с тестами сервиса по бронированию билетов в кино. Тесты запускаются локально и в Jenkins.

## Технологический стек

- **Java** 11.0.28
- **Cypress** 24.13.0
- **Jenkins** 2.426.1
- Браузеры для тестирования:
  - Google Chrome 144.0.7559.134 (stable)
  - Mozilla Firefox 147.0.3 (stable)

## Требования к окружению

- Установленная Java 11 (JDK)
- Установленный Node.js (версия, совместимая с Cypress 24.13.0)
- Установленные браузеры:
  - Google Chrome (stable)
  - Mozilla Firefox (stable)
- Git
